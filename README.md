Hotel Room Management System 
​Welcome to my first Python GUI project! This application is a fully functional system designed to manage business operations for a hotel.
​Features:
​Customer Onboarding: Seamlessly add and validate guest details.
​Booking Management: Select room types and services like Spa or Laundry with automated price calculation.
​Automated Billing: Generate total bills and invoices in real-time.
​Transaction Tracking: Maintain a complete history of payments and invoices.
​Tech Stack:
​Language: Python 
​Library: Tkinter (for the Graphical User Interface)
​Data Storage: CSV files (for bookings, customers, and payments)
​Project Files:
​fop.py: The main application logic and GUI code.
​customers.csv: Stores guest information.
​payments.csv: Tracks all financial transactions.
​active_bookings.csv: Manages current room statuses.
